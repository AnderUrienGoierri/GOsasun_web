# Abstract

This project presents GOsasun, a multiplatform healthcare management application whose main purpose is to improve the way patients are monitored in a health centre. The core of the system is the patient and their clinical follow-ups, which include blood pressure readings from tensiometers, pulse rate, weight, height, and clinical notes. In addition, relevant medical documents such as test results or reports can be attached to each follow-up record. The platform was developed using two complementary technologies: a PHP-based web application with jQuery for the online interface, and a C# desktop application that connects directly with real tensiometers to capture blood pressure and pulse data through XML. The system uses a MySQL relational database with views, triggers, and indexes to ensure data integrity and efficient queries. Three user roles were implemented: patients, who can enter their own measurements and view their history; healthcare workers, who manage follow-ups, prescriptions, and appointments for their assigned patients; and receptionists, who handle all administrative tasks including user management and scheduling. The application also supports multilingual configuration through XML files and provides graphical charts to visualise patient data over time. The final result is a functional and secure healthcare portal that facilitates patient self-monitoring and improves communication between all the members of the health centre.
